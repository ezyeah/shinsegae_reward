# Button 공통규칙

  • [태그 구분] 버튼 기능에 따라 사용 필요

    * 버튼 타입 전체에 해당되는 사항입니다.

    • a 태그 - 링크성 버튼일 경우 사용

      ┗ 비활성화시 : .disabled 클래스 추가

  • 버튼 class는 반드시 버튼 사이즈 클래스 1개 + color 클래스 1개의 조합으로 작업한다.

  • [ size 네이밍 정의 ]

    • btn-lg / btn-round-lg : height: 54 / font-size: 15px / font-weight: 600

    • btn-md / btn-round-md : height: 44 / font-size: 14px / font-weight: 600

    • btn-sm / btn-round-sm : height: 36 /font-size: 13px / font-weight: 600

    • btn-xs / btn-round-xs : height: 32 /font-size: 14px / font-weight: 400

    • btn-2xs / btn-round-2xs : height: 30 /font-size: 13px / font-weight: 500

  • [ color : 클래스 필수 지정 ]

    • blue : border, background : $colorBlue100 (#6987A1)

    • light-blue : border, background : $colorBlue10 (#F0F4F5)

    • gray90 : border, background : $colorGray90 (#333)

    • gray20 : border, background : $colorGray20 (#F0F0F0)

    • line-blue : border : $colorBlue100 (#6987A1) background : $colorGray00(#FFF)

    • line-brown : $colorBrown100 (#B09785) background : $colorGray00(#FFF)

    • line-gray40 : $colorGray40 (#DDD) background : $colorGray00(#FFF)

  • [ text + icon ]

  • [ btn-wrap : 버튼 양 옆 여백 ]

    • btn-wrap : 20px

    • btn-wrap .bottom: 버튼이 바닥에 위치 할 경우 margin -5px을 가짐
