# Button 공통규칙

  • [태그 구분] 버튼 기능에 따라 사용 필요

    * 버튼 타입 전체에 해당되는 사항입니다.

    • a 태그 - 링크성 버튼일 경우 사용

      ┗ 비활성화시 : .disabled 클래스 추가

  • 버튼 class는 반드시 버튼 사이즈 클래스 1개 + color 클래스 1개의 조합으로 작업한다.

  • [ size 네이밍 정의 ]

    • btn-lg / btn-square-lg : height: 54 / font-size: 15px / font-weight: 600

    • btn-md / btn-square-md : height: 44 / font-size: 14px / font-weight: 600

    • btn-sm / btn-square-sm : height: 36 /font-size: 13px / font-weight: 600

    • btn-xs / btn-square-xs : height: 32 /font-size: 14px / font-weight: 400

    • btn-2xs / btn-square-2xs : height: 30 /font-size: 13px / font-weight: 500

  • [ color : 클래스 필수 지정 ]

    • blue100 background, border : $colorBlue100 (#6987A1)

    • blue10 background, border : $colorBlue10 (#F0F4F5)

    • gray90 background, border : $colorGray90 (#333)

    • gray20 background, border : $colorGray20 (#F0F0F0)

    • line-blue background : $colorGray00(#FFF) / border : $colorBlue100 (#6987A1)

    • line-brown background : $colorGray00(#FFF) / border : $colorBrown100 (#B09785)

    • line-gray40 background : $colorGray00(#FFF) / border: $colorGray40 (#DDD)

  • 그 외 text, icon button

  • 버튼 정렬 (btn-wrap)

    • btn-wrap : 기본

    • btn-wrap-md : 상단 여백 20px, 버튼 사이 여백 6px

    • btn-wrap-lg : 상단 여백 30px, 버튼 사이 여백 6px

    • btn-wrap.row : 세로 정렬, 버튼 사이 여백 10px

    *페이지 최하단에 위치하는 경우 margin-left/right: -5px 가짐
