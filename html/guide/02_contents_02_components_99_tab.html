# Tab 공통규칙
  1. jquery UI - Tabs 사용
    - API Url : https://api.jqueryui.com/tabs/
  2. 기본 구조

    tab-wrap
    > .swiper-auto-wrap > tab-{$type}-list > tab-item > tab-link > tab-text
    > tab-content > content box

    * 균등형일 경우 .swiper-auto-wrap 구조 생략

  3. tab 기능 구분하여 tab-wrap 영역에 multi class 사용 필요
    • default 기본 tab
    • link-tab 링크 이동 tab
  4. tab type 구분하여 tab-{$type}-list class 사용 필요
    • tab-line-list 라인 type
    • tab-pill-list 알약 type
    • option
      • tab 균등 정렬 필요한 경우 tab-{$type}-list 영역에 justify-tab multi class 사용 필요
    • tab-box-list 박스 type

link-tab
  • 탭 선택 시 페이지 이동되어야 하는 경우 사용
  • 기존 tab-{$type}-list 영역에 link-tab multi class로 사용 필요
  • tab content 하나만 필요하므로 a.tab-link의 href 속성 값과 div.tab-content의 id 값을 맵핑할 필요가 없음

    ┗ <a href="javascript:" class="tab-link"></a>

    ┗ <div class="tab-content"></div> (id 값 필요 X)
