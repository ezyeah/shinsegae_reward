# Badge 공통규칙
  • badge : size + color 클래스 조합을 사용하여 작업한다.

  • [ badge 사이즈 네이밍 정의 ]

    • badge-lg, badge-round-lg : height: 26 / font-size: 13px / font-weight: 400

    • badge-md, badge-round-md : height: 22 / font-size: 12px / font-weight: 400

    • badge-sm, badge-round-sm : height: 16 / font-size: 11px / font-weight: 600

  • [ color : 클래스 필수 지정 ]

    • gray00 background, border : $colorGray00 (#000000)

    • gray20 background, border : $colorGray20 (#F0F0F0)

    • blue100 background, border : $colorBlue100 (#6987A1)

    • blue120 background, border : $colorBlue120 (#536F86)

    • brown100 background, border : $colorBrown100 (#B09785)

    • gray90 background : $colorGray90(#333) / border : $colorGray90 (#333)

    • line-gray40 background : $colorGray00(#FFF) / border: $colorGray40 (#DDD)

  • 뱃지 정렬 (badge-wrap)

    • badge-wrap-lg (뱃지 사이 간격 : 4px / 하단 여백 : 10px)

    • badge-wrap-md (뱃지 사이 간격 : 2px / 하단 여백 : 10px)

  • 태그

    • tag

    • tag-wrap
