# Popup 공통규칙
  • jquery UI - Dialog 사용
    - API Url : https://api.jqueryui.com/dialog/
  • dialog open/close
    - open : onclick="rewardPub.front.mo.dialogOnOff().popOpen('#popEx1');"
    - close : onclick="rewardPub.front.mo.dialogOnOff().popClose('#popEx1');"
  • 팝업 커스텀 시
    - 해당 팝업에만 css 적용 필요 시 class 네이밍 규칙 : .dialog+팝업명 (ex. dialog-search)
  • script 실행을 위한 dialog 필수 소스
    - popup DOM 구조 추가시, 고유 팝업 ID 필요 (버튼과 매핑하여 팝업 호출 시 사용)

    ┗ 팝업 ID 네이밍 규칙 : #pop+팝업명 (ex. popSearch)

  • layer popup type

    • popup 유형별 data-class 속성 값

      • default
      • dialog-full: full popup
      • dialog-bottom : bottom popup
      • dialog-alert : alert popup
      • dim-close : 딤 영역 클릭 시 닫기
Default(중앙팝업)
