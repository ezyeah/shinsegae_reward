input

  • input은 타입별로 구분하여 사용하며, 공통 class인 .input-box에 해당하는 multi class를 조합하여 사용한다.

  • [input 타입별 multi class 지정]

    • 검색 : .form-input-search

    • 유효성 검사 : .valid-check * 유효성 검사 결과 case별 class 제어 필요 (valid-check 영역에 multi class로 추가)

      ┗ class 추가 없음 : 기본(유효성검사 결과 문구 미노출)

      ┗ fail : 실패

    • [ .ui-input multi class 지정 ]

      • 입력 텍스트 오른쪽 정렬 : .right
