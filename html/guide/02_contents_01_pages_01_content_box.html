Content Box Guide
# Content Box 공통규칙
  • cont type, top size 구분하여 class 사용 필요
    • {$type}cont-box-{$size}의 형태로 조합

      ㄴ type gray, top gap 20px 필요한 경우 : gcont-md-box

    • cont type
      • cont bg-color: #FFF
      • gcont bg-color: #F5F5F5
      • rcont bg-color: #F0F0F0 / border-radius: 16px
  • top size
    • zero padding-top: 0/padding-bottom: 30px *임시
    • xs padding-top/bottom: 10px
    • sm padding-top/bottom: 16px
    • md padding-top/bottom: 20px
    • lg padding-top/bottom: 30px
index
  1. cont type
  2. top size
  3. Content Inner
